**BMI Calculator Web App**
- This is a simple Body Mass Index (BMI) Calculator web application built using HTML, CSS, and JavaScript.
- It allows users to input their height (in feet and inches) and weight (in kilograms) to calculate their BMI.
- The app also displays a table showing the different BMI categories and their corresponding ranges.

**Features:**
- Height Input: Allows users to select their height in feet and inches.
- Weight Input: Allows users to input their weight in kilograms.
- BMI Calculation: Automatically calculates and displays the user's BMI based on the inputs.
- BMI Categories: Displays the BMI categories (Underweight, Healthy, Overweight, and Obesity) with their corresponding ranges.
- Responsive UI: The layout is responsive, ensuring a good user experience across various devices.
- Interactive Reset Button: A reset button is available to clear the input fields.

**Technologies Used:**
- HTML5: For structuring the web page.
- CSS3: For styling and layout.
- JavaScript: For dynamic calculations and interactivity.

**How to Use:**
- Enter your height (in feet and inches) and weight (in kilograms).
- Click on the "Calculate Your BMI" button to get your BMI.
- The result will be displayed and categorized according to the BMI range.
- Use the "Reset" button to clear the input fields.
